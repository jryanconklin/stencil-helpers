## Spacing Properties

Use the `py` and `px` properties to set horizontal and vertical spacing on a content box. Defaults to no spacing and only
a limited set of sm-xl are available. Example: `<frog-content-box py="sm" px="sm"></frog-content-box>`.

## Display Variants

### Shadow

Apply the `shadow` variant to wrap content in a shadow box with slight hover effect. Example: `<frog-content-box variant="shadow"></frog-content-box>`

You can pass grid items to the box to wrap grid content.
